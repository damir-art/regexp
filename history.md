# История регулярных выражений
- Появились в 1943-м в области нейронауки
- 1956 появились регулярные выражения
- 1968 вошли в Unix
- grep, грепнуть - найти что-то с помощью регулярных выражений
- 1986, POSIX - новый стандарт спроектированный для того чтобы удостовериться в совместимости между различными операционными системами
  - BREs - базовые регулярные выражения (grep)
  - EREs - расширенные регулярные выражения (egrep)
- 1987, Perl - улучшает регулярки делая его стандартом для всех языков
